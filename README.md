# brave-goggles

Personal Brave Search Goggle for customizing search result rankings.

## Usage

Submit the raw URL to Brave:
```
https://raw.githubusercontent.com/jjc-/brave-goggles/main/my_goggle.goggle
```

## Resources

- [Goggles Quickstart](https://github.com/brave/goggles-quickstart) - Official documentation and syntax reference
- [Discover Goggles](https://search.brave.com/goggles/discover) - Browse community goggles
